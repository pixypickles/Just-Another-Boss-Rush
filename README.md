# ありきたりなボス5戦 — GitHub Pages版 v6

## 3戦目以降の強化
- ケルベロス
  - 高速突進
  - 追尾する炎弾
  - 高威力の近距離攻撃
- ドラゴン
  - 手下召喚
  - 一定時間残る火柱
  - 追尾弾
  - 火炎弾
- 魔王
  - 高耐久・高攻撃力
  - 複数の追尾弾
  - 強化手下召喚
  - 長時間残る煉獄の火柱
  - 高威力衝撃波
  - HP50%・25%以下で行動間隔と攻撃数が強化

火柱と高威力衝撃波は、盾だけでは完全に防げません。赤い予告範囲から移動して回避してください。

## GitHubへの反映
`index.html`、`style.css`、`game.js`、`.nojekyll` をリポジトリのルートへ上書きしてください。
古い動作が残る場合は、公開URL末尾に `?v=6` を付けてください。
