# Just Another Boss Rush v64.1

## 武神挑戦モード 第1段階

解放条件（いずれか）:
- 覚醒無双ではない1人モードで本編クリア
- ドラキュラ・竜騎士・模倣術師の3人で本編クリア

実装:
- 1〜3人の武神挑戦モード（人数別HP補正）
- 小型の武神ボス
- パンチ残像近接
- ジャンプ中無敵
- ジャンプ後の斜め急降下蹴り
- 薄い赤ライン予告からの神速飛び蹴り（最大HP95%級ダメージ）

カウンター構えは次段階です。


## v64.1
- 武神の飛び蹴り画像を、実際の移動方向に合わせて左右反転。
- 青いオーラのカウンター構えを追加。構え中は攻撃・飛び道具・状態異常を無効化。
- 構え中に攻撃されると一瞬消え、上空から斜め急降下蹴りで反撃。
