# ありきたりなボス5戦 — GitHub Pages版 v9

## 更新内容
- 1人モードでナイト以外へ選び直せない問題を修正
- 人数分を選択済みの状態で別キャラを押すと、選択キャラを入れ替える方式へ変更
- ナイトのセルフヒールを82から190へ強化
- ナイトの円形バリア中は毎秒32HPずつ回復
- ナイトとヒーラーの移動速度を上昇
- トロールとの密着時に互いを押し離し、接触ダメージへ間隔を設定
- 接触時のノックバック方向を修正し、ボスから離れる方向へ押し出す
- ケルベロス、ドラゴン、魔王が手下を召喚
- 手下は追跡・接触攻撃を行い、主人公の攻撃で倒せる
- 魔王はHP低下時に強化手下の召喚数が増加

## GitHubへの反映
`index.html`、`style.css`、`game.js`、`.nojekyll` をリポジトリのルートへ上書きしてください。
古い動作が残る場合は、公開URL末尾に `?v=9` を付けてください。
