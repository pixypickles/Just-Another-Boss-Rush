# Just Another Boss Rush v61

## 模倣術師 第2段階

- 3人編成：模倣術師を除く仲間1人目のA、仲間2人目のBを模倣
- 2人編成：同じ仲間のAとBを模倣
- 1人編成：Aは巨棍撃、Bは竜散弾
- コピー対象は単純な近接、飛び道具、簡易突進に限定
- 回復、設置、変身などコピー不可技は安全な代替技へ変更
- ドラゴンチャージの模倣は短押し版のみ
- CとDは第1段階の仮仕様を維持

## 解放条件修正

- 気功師：モンク単独クリア（通常・覚醒無双どちらも可）
- 忍者：覚醒無双以外を1人または2人でクリア
