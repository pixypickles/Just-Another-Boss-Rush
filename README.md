# Just Another Boss Rush v63

## v63 模倣術師・ボス別ビルド

- 模倣術師を操作キャラにした2人・3人編成では、戦闘開始前に5ボス分のA/B模倣技を1画面で設定
- 仲間が持つコピー可能なA/B/C技から自由選択
- 同じ技をA/Bへ重複設定可能。クールタイムは各ボタンで独立
- 技名とコピー元キャラ名を表示
- 設定はパーティ構成ごとにブラウザへ保存
- 気功壁はA/Bそれぞれ1枚ずつ設置可能
- ドラゴンチャージ模倣版は短押しのみ
- ソロ時は従来どおり巨棍撃／竜散弾

# Just Another Boss Rush v62

## 模倣術師 第3段階（C固有技）

- Cを「獣化・双牙突進」へ変更
- 1回目はボス脇へ高速突進
- 約0.1秒後、鋭角に切り返してボスへ2回目の突進
- 突進中は短時間無敵、敵弾を消去、手下を貫通
- 2回の軌道へ短時間だけ3本爪痕を表示
- 爪痕は既存の斬撃配列を使うため、追加画像・常駐処理なし

A/Bコピー機能と既存の解放条件はv61のまま維持しています。
