## v32 覚醒ルーン・手下変換
- オーバーロード中に設置したルーンは、手下が踏んでも消滅しないよう変更。
- 手下が覚醒ルーンを踏むと、その場で同じ色の対ボス追尾弾へ変換。
- 変換弾は炎・氷・雷すべて共通性能・共通ダメージで、色だけ元ルーンを継承。
- 変換弾は手下に当たらず、ボスだけを追尾して命中。
- キャッシュ更新用の読み込みバージョンを v32 に更新。


## v33 気功師・チャージシステム
- 新キャラクター「気功師」を追加。赤い中華道着と黒髪のグラフィックを採用。
- A「集束気功弾」：最大3秒チャージ。威力・弾サイズが上昇し、敵弾を消去。
- B「拡散気功弾」：最大4秒チャージ。規則正しい扇状に発射し、チャージで弾数・列数・中央密度が上昇。
- C「爆芯掌」：チャージ上限なし。手の先を中心とする5段階の同心円を表示し、中心ほど大ダメージ。30秒の中心命中でボス一撃級。
- 気功師はチャージ中、被ダメージが2倍。ボタンを離した瞬間に発動し、チャージの保存は不可。
- 1人専用スキル「気脈解放」：10秒間、A・Bは押すだけで最大チャージ発射、Cは5倍速チャージ。
- ルーンメイジはオーバーロード中、A/B/Cを押しっぱなしで連続設置可能。
- キャッシュ更新用の読み込みバージョンを v33 に更新。
