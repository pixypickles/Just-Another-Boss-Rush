# ありきたりなボス5戦 — GitHub Pages版 v8

## 修正内容
- 編成後に「封印の間」の表示で止まる問題を修正
- 戦闘開始前は、存在しないボスや主人公を描画しないよう変更
- 編成後に主人公とボスを生成して、すぐ戦闘開始
- 細長い縦画面では、ゲーム画面を横幅と同じ高さの正方形に近い比率で中央配置
- 通常のスマホや折りたたみスマホでは、可能な範囲で縦横比を1:1へ近づける

## GitHubへの反映
`index.html`、`style.css`、`game.js`、`.nojekyll` をリポジトリのルートへ上書きしてください。

古い表示が残る場合は、公開URLの末尾に `?v=8` を付けてください。
